# 📡 Smart Street Light Monitoring System (GSM-Based)

A full-stack IoT-enabled smart street lighting system designed to monitor and manage street lights remotely using GSM communication. This project leverages a modern tech stack with a robust backend, responsive frontend, and reliable database integration to enable efficient urban infrastructure management.

---

## 🚀 Key Features

- **GSM-Based Communication**: Street light nodes send real-time status updates (ON/OFF, faults, power usage) via GSM modules.  
- **MQTT Integration**: Lightweight messaging protocol for efficient, real-time IoT communication and device coordination.  
- **Remote Monitoring**: Centralized dashboard to monitor all street lights from anywhere.  
- **Fault Detection & Alerts**: Automatically detects issues like light failure or power anomalies and notifies administrators.  
- **Energy Efficiency Insights**: Tracks usage patterns to help optimize power consumption.  
- **User-Friendly Interface**: Interactive and responsive UI built with React.  
- **Scalable Backend**: RESTful APIs developed using Spring Boot for seamless data handling.  
- **Reliable Data Storage**: PostgreSQL database ensures structured and persistent data management.  

---

## 🛠️ Tech Stack

- **Backend**: Spring Boot (Java)  
- **Frontend**: React.js  
- **Database**: PostgreSQL  
- **Communication**: GSM Module + MQTT Protocol  

---

## 📊 Use Cases

- Smart city infrastructure  
- Energy-efficient street lighting systems  
- Municipal monitoring and maintenance automation  

---

## 🔧 System Architecture

The system consists of GSM-enabled street light controllers that transmit data to a central server using GSM and MQTT protocols. The backend processes and stores this data in PostgreSQL, while the React frontend provides a real-time visualization dashboard for users.
